# Worldmonitor Hourly Service

## Overview

Hourly news data collection and caching service for worldmonitor.

## Tech Stack

- **Language**: JavaScript/Node.js

## Startup

```bash
cd worldmonitor-hourly
docker compose up -d
```

## Project Structure

```
worldmonitor-hourly/
├── config/                  # Configuration
├── data/                    # Data output (hourly cache)
├── scripts/                # Scripts
└── docker-compose.yml
```
