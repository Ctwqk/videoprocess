# News Publisher Service

## Overview

News publishing and distribution platform.

## Tech Stack

- **Language**: JavaScript/Node.js

## Startup

```bash
cd news-publisher
docker compose up -d
```

## Project Structure

```
news-publisher/
├── dashboard/              # Dashboard UI
├── publishers/            # Publisher implementations
├── data/                  # Data storage
├── scripts/               # Utility scripts
└── docker-compose.yml
```
