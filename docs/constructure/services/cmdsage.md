# Cmdsage Service

## Overview

Command-line sage tool for enhanced CLI productivity.

## Tech Stack

- **Language**: Rust

## Startup

```bash
cd cmdsage
cargo build --release
./target/release/cmdsage
```

## Project Structure

```
cmdsage/
├── src/                   # Rust source
├── commands/              # Command implementations
├── models/                # Data models
├── cache/                 # Caching layer
└── Cargo.toml
```
